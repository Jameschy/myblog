phome_enewsfile 数据表字段解释（附件表）

字段名 类型 解释 附加说明
fileid int(11) 附件ID
filename varchar(60) 文件名
filesize int(11) 文件大小
path varchar(20) 存放日期目录
adduser varchar(30) 上传者
filetime datetime 上传时间
classid smallint(6) 栏目ID
no varchar(60) 文件编号
type tinyint(4) 文件类型 1为图片，2为Flash文件，3为多媒体文件，0为附件
onclick int(11) 下载次数
id bigint(20) 信息ID
cjid bigint(20) 文件临时识别编号 值0
fpath tinyint(1) 文件存放目录方式 0为栏目目录，1为/d/file/p目录，2为/d/file目录
